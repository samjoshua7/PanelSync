## 🚀 Overview

**Panel Sync** is a web-based system designed to synchronize content across multiple screens in real-time.  
It is ideal for digital signage, presentations, and remote display control systems.

---

## ✨ Features

- 🔄 Real-time synchronization across multiple screens  
- 📺 Remote screen control using unique screen IDs  
- ⚡ Fast and responsive UI  
- ☁️ Firebase-powered backend integration  
- 🧩 Dynamic content updates  
- 🌐 Cross-device compatibility (TV, PC, Mobile)

## ⚙️ Installation

### Clone the repository
git clone https://github.com/your-username/panelsync.git
cd panelsync

### Install dependencies
npm install

### Setup environment variables
Create a `.env` file:

VITE_API_URL=your_backend_url
FIREBASE_API_KEY=your_key
FIREBASE_PROJECT_ID=your_project

### Run locally
npm run dev
